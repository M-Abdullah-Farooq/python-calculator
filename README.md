# Simple Calculator

A beginner-friendly calculator built using Python that does basic Arithmetic operations such as:
* Addition
* Subtraction
* Multiplication
* Division
* ## Special Features
* Input validation
* Division-by-zero handling
* Continuous calculations until the user chooses to quit
## What I Learned

* Loops ('while')
* Conditional statements (`if/elif/else`)
* Exception handling (`try/except`)
* User input validation
* Basic program structure
* Nested Loop Structure

## Future Improvements

* Scientific calculator functions
* Percentage and power operations
* Doing operation on n numbers instead of jsut two.
